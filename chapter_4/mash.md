# Mash

Mash 是一个使用 MinHash 方法来做快速的基因组与元基因组距离估计的软件。

## 安装 Mash

```bash
~/apps$ wget https://github.com/marbl/Mash/releases/download/v1.0.2/mash-Linux64-v1.0.2.tar.gz
~/apps$ tar zxf mash-Linux64-v1.0.2 -C Mash
```
